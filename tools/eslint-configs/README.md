# Shared ESLint Configs

## Usage

```json
{
  "extends": [
    "../../../../.eslintrc.json",
    "../../../../tools/eslint-configs/base.js",
    "../../../../tools/eslint-configs/+jest.js"
  ]
}
```

## Presets

### `nest.js`

For NestJS.

### `react.js`

For React.

### `e2e.js`

For Cypress E2E App.

## Addon

### `+jest.js`

For Jest.
